A — Three-Panel Shell

Left sidebar holds session config + history. Center is voice + transcript. Right panel shows corrections.

Pros

  • Everything visible at once
  • Desktop-app feel
  • History always accessible

Cons

  • Cramped on smaller screens
  • Corrections compete for attention mid-conversation

B — Chat + Tools Split

Left pane is a full-height chat with voice button pinned to bottom. Right pane stacks config (collapsible), corrections, and history.

Pros

  • Familiar chat-app pattern
  • Transcript gets the most space
  • Config stays out of the way

Cons

  • Right pane can get crowded
  • Less immersive voice feel

C — Immersive Voice Mode + Drawer

Full-screen dark mode voice experience. Transcript floats above the mic button. Config and history slide up from a bottom drawer.

Pros

  • Voice-first — nothing competes with the mic
  • Feels native on Android
  • Minimal distraction during conversation

Cons

  • Corrections and config require extra tap
  • Less scannable on desktop
